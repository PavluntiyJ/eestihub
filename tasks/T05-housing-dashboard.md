> после приёмки T03/T04. Ниже — рамка скоупа.

**Прочитай сначала:** `docs/CONTEXT.md` (§5 — контракт `GET /api/v1/housing/rents`).
**Зависимости:** T03, T04 приняты.

## Рамка

Backend:
- `app/services/housing_service.py` — выдаёт мок-данные по районам Таллина
  (Kesklinn, Põhja-Tallinn, Kristiine, Mustamäe, Lasnamäe, Haabersti,
  Nõmme, Pirita); структура — по контракту §5. Данные лежат отдельным
  модулем/JSON, чтобы позже заменить на реальный парсер без смены API.
- `app/schemas/housing.py`, `app/api/v1/routes/housing.py`, тесты.

Frontend:
- Страница `/[locale]/housing` (все строки — через словари i18n, en/et/ru):
  дашборд средних цен — таблица районов + визуализация
  (bar chart по районам; библиотеку графиков предложит оркестратор в
  детальном брифе). Серверный fetch, интерактив — только в клиентских
  листовых компонентах.

Детальные критерии приёмки появятся в этом файле позже.
